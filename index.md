# Exploitation Report

Current threat activity reveals a sophisticated landscape of active exploitation campaigns targeting enterprise environments and critical infrastructure. Notable threat actors including RomCom, ToddyCat, and North Korean-linked groups are deploying advanced techniques ranging from fake update mechanisms to specialized malware frameworks. A critical vulnerability in Oracle Identity Manager is under active exploitation, while attackers continue to leverage social engineering through fake Windows updates, compromised 3D model files, and business email compromise tactics. The ShadowRay 2.0 campaign demonstrates how threat actors are hijacking AI infrastructure for cryptocurrency mining operations, while commercial spyware campaigns are targeting high-value messaging platform users.

## Active Exploitation Details

### Oracle Identity Manager Critical Vulnerability
- **Description**: A critical flaw in Oracle Identity Manager allowing unauthorized access to enterprise identity management systems
- **Impact**: Attackers can potentially compromise identity management infrastructure and gain unauthorized access to corporate resources
- **Status**: Currently under active exploitation following previous Oracle Cloud breaches
- **CVE ID**: CVE-2025-61757

### Ray Framework Vulnerability
- **Description**: A security flaw in the Ray framework used in AI infrastructure that enables remote access and code execution
- **Impact**: Threat actors can hijack AI clusters, deploy cryptocurrency mining operations, and steal sensitive data through self-propagating botnets
- **Status**: Actively exploited by the ShadowRay 2.0 campaign to compromise AI infrastructure worldwide

### Fake Update Exploitation Methods
- **Description**: Multiple campaigns using fake Windows Update screens and browser-based update notifications to deliver malware
- **Impact**: Users are tricked into executing malicious commands, leading to information stealer deployment and system compromise
- **Status**: Active campaigns including JackFix and ClickFix variants targeting users through adult websites and realistic update animations

## Affected Systems and Products

- **Oracle Identity Manager**: Enterprise identity management systems vulnerable to unauthorized access
- **Ray Framework**: AI infrastructure and machine learning clusters worldwide
- **Windows Systems**: Targeted through fake update campaigns and social engineering
- **macOS Environments**: Specifically targeted by North Korean FlexibleFerret campaign
- **Microsoft 365 and Outlook**: Email systems targeted by ToddyCat threat actor tools
- **Signal and WhatsApp**: Messaging platforms targeted by commercial spyware campaigns
- **Blender 3D Software**: Users targeted through malicious model files on platforms like CGTrader
- **Oracle E-Business Suite**: Systems targeted by Clop extortion gang attacks
- **OnSolve CodeRED Platform**: Emergency notification systems disrupted by cyberattacks

## Attack Vectors and Techniques

- **Fake Update Campaigns**: Browser-based fake Windows Update screens with hidden malicious code in images
- **SocGholish JavaScript Loader**: Used by RomCom to deliver Mythic Agent malware to civil engineering companies
- **Malicious 3D Model Files**: StealC V2 information stealer distributed through compromised Blender files
- **Business Email Compromise**: Account takeover fraud resulting in $262 million in losses since January
- **Social Engineering**: FlexibleFerret campaign using "Contagious Interview" tactics against macOS users
- **ClickFix Variants**: JackFix attacks circumventing traditional ClickFix mitigations with increased psychological pressure
- **Commercial Spyware**: CISA-identified campaigns targeting high-value Signal and WhatsApp users
- **Supply Chain Attacks**: Shai-hulud worm variant executing malicious code during software preinstall phases

## Threat Actor Activities

- **RomCom**: Targeting U.S. civil engineering companies using SocGholish fake update attacks to deploy Mythic Agent malware
- **ToddyCat**: Deploying new hacking tools including TCSectorCopy to steal Outlook emails and Microsoft 365 access tokens
- **North Korean FlexibleFerret**: Continuing "Contagious Interview" campaign refinements to target macOS users and steal credentials
- **Chinese State-Linked Groups**: Conducting surveillance operations against Russian IT organizations using commercial cloud services
- **Clop Extortion Gang**: Attacking Oracle E-Business Suite systems including Dartmouth College data breach
- **Iranian Cyber Operations**: Deploying "cyber-enabled kinetic targeting" in coordination with missile attacks
- **ShadowRay 2.0 Operators**: Hijacking AI infrastructure worldwide for cryptocurrency mining and data theft operations
- **Russian-Linked Campaigns**: Distributing StealC V2 information stealer through malicious Blender 3D model files
